## How To Use

```bash
# Clone this repository
$ git clone https://github.com/cqrlosArch/webpack-workflow-2021

# Install dependencies
$ yarn install or npm install

# Run mode development
$ yarn start or npm start

# Build app
$ yarn build or npm run build
```
